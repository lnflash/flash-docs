# Flash 연락처 관리

연락처를 효율적으로 관리하면 결제 전송, 채팅, Flash 네트워크 구축이 쉬워집니다. Flash의 강력한 연락처 시스템을 최대한 활용하는 방법을 알아보세요.

## 🚀 프로 팁

### 효율성 해킹
1. **키보드 단축키**
   - `Cmd/Ctrl + K`: 빠른 검색
   - `Space`: 연락처 미리보기
   - `S`: 빠른 전송

2. **제스처**
   - 스와이프하여 결제
   - 길게 눌러 옵션
   - 핀치하여 그룹화

3. **자동화**
   ```javascript
   const automations = [

     "비활성 연락처 정리",

   ];
   ```
