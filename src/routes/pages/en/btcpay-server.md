## BTCPay Server

Flash used to rely on BTCPay Server for some payment flows.

That integration is now legacy. If you are looking for today's user flows, see:

- [Cash Wallet](guides/cash-wallet)
- [Bitcoin Wallet](guides/bitcoin-wallet)
- [Swap Between Wallets](guides/swap)
- [Top Up](guides/top-up)
- [Cash Out](guides/cash-out)

## What to know

- BTCPay is not the main Flash user experience anymore
- Most users do not need to interact with BTCPay directly
- If you need help with a current payment flow, contact Flash support
