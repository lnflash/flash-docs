## Flash is Open Source

Flash is committed to building on open, transparent technology. Our code is publicly available and we actively contribute to the Bitcoin open-source ecosystem.

## Flash Repositories

- [flash-mobile](https://github.com/lnflash/flash-mobile) — The Flash mobile app for iOS and Android
- [flash](https://github.com/lnflash/flash) — Flash backend (Bitcoin banking infrastructure)
- [flash-docs](https://github.com/lnflash/flash-docs) — This documentation site
- [flash-pos](https://github.com/lnflash/flash-pos) — Flash POS app for merchants
- [pulse](https://github.com/lnflash/pulse) — Flash's AI agent for WhatsApp payments

## Technologies We Build On

Flash wouldn't exist without these open-source projects:

- **[Bitcoin Core](https://bitcoin.org/en/bitcoin-core/)** — The reference implementation of Bitcoin
- **[Lightning Network](https://lightning.network/)** — Instant, low-fee Bitcoin payments
- **[Breez SDK](https://sdk.breez.technology/)** — Lightning wallet toolkit powering Flash's BTC wallet
- **[Galoy](https://galoy.io/)** — Open-source Bitcoin banking infrastructure
- **[IBEX Mercado](https://www.ibexmercado.com/)** — Lightning Network infrastructure
