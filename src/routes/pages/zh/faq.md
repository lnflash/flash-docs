## 常见问题

快速查找使用Flash的常见问题答案。找不到您要的答案？请通过[support@getflash.io](mailto:support@getflash.io)联系我们。

### 开始使用

<details class="faq-item">
<summary><strong>什么是Flash？</strong></summary>

Flash是专门为加勒比地区创建产品和服务的比特币服务提供商。我们的生态系统包括供日常用户使用的Flash应用程序、供商家使用的Flashpoint POS设备、用于奖励和支付的Flashcard，以及无缝的比特币充值和现金结算。

</details>

<details class="faq-item">
<summary><strong>Flash在哪些国家/地区可用？</strong></summary>

Flash目前在牙买加可用，并正在整个加勒比地区扩展。查看我们的[地图](https://map.flashapp.me)查找您附近的Flash位置。

</details>

<details class="faq-item">
<summary><strong>个人账户、专业账户和商家账户有什么区别？</strong></summary>

<ul>
<li><strong>个人账户</strong>：个人使用的基本比特币和美元钱包</li>
<li><strong>专业账户</strong>：增强功能，包括银行转账（仅限牙买加）</li>
<li><strong>商家账户</strong>：商业功能，包括Flashpoint POS集成和奖励计划</li>
</ul>
</details>

<details class="faq-item">
<summary><strong>如何开始使用Flash？</strong></summary>

<ol>
<li>从应用商店下载Flash应用</li>
<li>按下"开始"创建账户，或使用您的电话号码或电子邮件登录现有账户</li>
<li>升级您的账户（如果进行银行转账）</li>
<li>通过银行转账、现金存款或比特币转账为您的钱包充值</li>
<li>开始发送和接收付款！</li>
</ol>

查看我们的[入门指南](/zh/get-started)了解详细说明。

</details>

### 账户与安全

<details class="faq-item">
<summary><strong>如何重置我的PIN？</strong></summary>

<ol>
<li>在登录屏幕上，点击"忘记PIN？"</li>
<li>输入您的电话号码</li>
<li>通过短信验证码验证您的身份</li>
<li>创建新的6位PIN</li>
<li>确认您的新PIN</li>
</ol>

如需更多帮助，请联系[support@getflash.io](mailto:support@getflash.io)。

</details>

<details class="faq-item">
<summary><strong>我的资金在Flash中安全吗？</strong></summary>

是的！Flash使用多层安全保护：

<ul>
<li>非托管比特币钱包（您控制密钥）</li>
<li>所有数据采用银行级加密</li>
<li>生物识别认证选项</li>
<li>定期安全审计</li>
<li>在运营管辖区的监管合规</li>
</ul>
</details>

<details class="faq-item">
<summary><strong>如果我丢失了手机会怎样？</strong></summary>

<ol>
<li>如果被盗，请立即联系支持人员冻结您的账户！</li>
<li>您的现金钱包资金仍然安全 - 它们不存储在您的手机上</li>
<li>如果启用，您的比特币钱包是非托管的，因此您控制密钥 - 请务必记下您的恢复短语！</li>
<li>在新设备上安装Flash</li>
<li>使用您的电话号码或电子邮件登录</li>
<li>如果您的比特币钱包已启用，请使用恢复短语恢复您的比特币钱包</li>
<li>设置新的PIN并重新启用生物识别认证</li>
<li>如果账户被盗，请联系支持人员并验证您的身份以解冻您的账户</li>
</ol>

专业提示：为比特币钱包启用生物识别认证并记下您的恢复短语。如果您的手机丢失或被盗，您可以快速重新访问您的资金，并应立即将其转移到新的比特币钱包。恶意行为者可能会尝试访问您的账户（如果他们有您的手机和PIN），因此最好迅速采取行动。

</details>

<details class="faq-item">
<summary><strong>我可以拥有多个Flash账户吗？</strong></summary>

每个电话号码只能关联一个Flash账户。如果您使用不同的电话号码或电子邮件，可以拥有多个账户，但我们建议每人只使用一个账户保持简单。
如果您需要管理多个账户，请考虑使用专业账户以获得增强功能，或使用商家账户满足业务需求。

</details>

### 支付与转账

<details class="faq-item">
<summary><strong>使用Flash的费用是多少？</strong></summary>

<ul>
<li><strong>应用内转账</strong>：Flash用户之间免费</li>
<li><strong>比特币网络费用</strong>：根据网络拥堵情况而定</li>
<li><strong>闪电网络费用</strong>：免费</li>
<li><strong>银行转账</strong>：Flash收取2%费用（加上接收银行收取的任何银行费用）</li>
<li><strong>现金结算</strong>：2%费用（因地点而异，可能会变更）</li>
<li><strong>商家处理</strong>：免费（因地点而异，可能会变更）</li>
</ul>

在确认交易前请务必查看费用。

</details>

<details class="faq-item">
<summary><strong>转账需要多长时间？</strong></summary>

<ul>
<li><strong>Flash到Flash</strong>：即时</li>
<li><strong>闪电转账</strong>：即时</li>
<li><strong>链上比特币转账</strong>：通常10-60分钟</li>
<li><strong>银行存款</strong>：0-1个工作日（牙买加）</li>
<li><strong>现金结算</strong>：在合作地点当天</li>
</ul>
</details>

<details class="faq-item">
<summary><strong>交易限额是多少？</strong></summary>

Flash限额是每日限额，根据账户类型和验证级别而异：

<ul>
<li><strong>试用账户</strong>：每天125美元</li>
<li><strong>个人账户</strong>：每天1,000美元</li>
<li><strong>专业账户</strong>：每天5,000美元</li>
<li><strong>商家账户</strong>：每天50,000美元</li>
</ul>

在应用中的设置 > 交易限额下查看您的当前限额。

</details>

<details class="faq-item">
<summary><strong>我可以取消交易吗？</strong></summary>

<ul>
<li><strong>待处理的链上交易</strong>：比特币钱包上的待处理链上交易可能可以退款到您选择的另一个链上比特币钱包 - 查看交易详情屏幕上的"可退款"按钮</li>
<li><strong>已完成的比特币转账</strong>：无法撤销（区块链不可变性）</li>
<li><strong>Flash到Flash转账</strong>：无法撤销。如有问题请立即联系支持人员</li>
<li><strong>失败的交易</strong>：在交易成功之前，资金永远不会离开您的钱包。这称为"原子"交易</li>
</ul>

_注意：比特币交易在设计上是不可撤销的。发送前请务必仔细检查收件人信息。_

</details>

### Flashcard

<details class="faq-item">
<summary><strong>如何获得Flashcard？</strong></summary>

<ol>
<li>访问任何参与的Flash商家</li>
<li>请求Flashcard</li>
<li>加载初始余额（可能适用最低金额）</li>
<li>将卡链接到您的Flash应用（可选）</li>
<li>开始在参与地点购买时赚取奖励！</li>
</ol>

查看我们的[Flashcard指南](/zh/guides/flashcard)了解详情。

</details>

<details class="faq-item">
<summary><strong>如果我的Flashcard丢失或被盗怎么办？</strong></summary>

<ol>
<li>立即打开Flash应用</li>
<li>转到卡片部分</li>
<li>选择您丢失的卡</li>
<li>点击"清除卡片"</li>
<li>如果您没有看到"清除卡片"选项，请联系支持人员将丢失卡片的资金"清除"到您的Flash钱包</li>
</ol>

_注意：清除卡片会将剩余余额转移到您的Flash钱包。该卡将被停用，无法再次使用。如果您尚未将卡片链接到Flash应用，我们可能无法清除卡片，因此请在收到卡片后立即将其链接到Flash应用。_

</details>

<details class="faq-item">
<summary><strong>Flashcard奖励如何运作？</strong></summary>

<ul>
<li>在Flashpoint商家的每次购买都能赚取积分</li>
<li>积分会自动添加到您的Flash账户</li>
<li>1积分 ~= 0.15牙买加元（或您当地货币的等值，可能会变更）</li>
<li>积分率因商家和促销而异</li>
<li>兑换积分获得在线礼品卡或商家优惠</li>
<li>只要您的卡处于活动状态，积分永不过期</li>
<li>在Flash应用中查看积分余额</li>
</ul>
</details>

### 技术问题

<details class="faq-item">
<summary><strong>应用无法打开或持续崩溃</strong></summary>

尝试以下步骤：

<ol>
<li>强制关闭应用并重新启动</li>
<li>在应用商店检查应用更新</li>
<li>重启您的手机</li>
<li>清除应用缓存（Android）或重新安装（iOS）</li>
<li>确保您的操作系统是最新的</li>
</ol>

仍有问题？联系[support@getflash.io](mailto:support@getflash.io)。

</details>

<details class="faq-item">
<summary><strong>我没有收到短信验证码</strong></summary>

<ol>
<li>检查您的手机是否有信号/数据连接</li>
<li>验证电话号码是否正确</li>
<li>检查短信是否被您的运营商阻止</li>
<li>60秒后尝试再次请求代码</li>
<li>如果问题持续存在，请联系支持人员</li>
</ol>
</details>

<details class="faq-item">
<summary><strong>交易失败或卡住</strong></summary>

常见解决方案：

<ul>
<li>检查您的余额是否足够（包括费用）</li>
<li>验证收件人详细信息是否正确</li>
<li>确保稳定的互联网连接</li>
<li>对于链上比特币交易：网络可能拥堵，等待并重试。（查看<a href=https://mempool.space>比特币区块链</a>查看网络是否拥堵）</li>
<li>检查应用通知以获取特定错误消息</li>
</ul>
</details>

<details class="faq-item">
<summary><strong>如何在我的企业接受Flash支付？</strong></summary>

<li>开始接受付款并提供奖励！</li>
</ol>

在我们的[商业部分](/zh/business)了解更多。

</details>

<details class="faq-item">
<summary><strong>接受Flash的好处是什么？</strong></summary>

<ul>
<li>比传统支付方式更低的交易费用</li>
<li>即时结算选项</li>
<li>访问比特币和美元</li>
<li>内置客户奖励计划</li>
<li>无退款</li>
<li>扩大客户群</li>
<li>详细的分析和报告</li>
</ul>
</details>

<details class="faq-item">
<summary><strong>商家结算流程如何运作？</strong></summary>

选择您喜欢的结算方式：

<ul>
<li><strong>即时比特币</strong>：立即接收比特币</li>
<li><strong>美元钱包</strong>：在Flash中保持美元资金</li>
<li><strong>银行转账</strong>：结算到您的银行账户（牙买加）</li>
<li><strong>现金提取</strong>：在合作地点可用</li>
</ul>

结算频率可以根据您的偏好设置为每日、每周或每月。在我们的[商业部分](/zh/business)了解更多。

</details>

### 比特币与闪电网络

<details class="faq-item">
<summary><strong>我是比特币新手。很复杂吗？</strong></summary>

一点也不！Flash让比特币像使用任何支付应用一样简单：

<ul>
<li>无需技术知识</li>
<li>以您的当地货币计价</li>
<li>Flash自动处理转换</li>
<li>与传统应用相同的用户体验</li>
<li>可选：在我们的[比特币基础](/zh/bitcoin-protocol)指南中了解更多</li>
</ul>
</details>

<details class="faq-item">
<summary><strong>比特币和闪电网络有什么区别？</strong></summary>

<ul>
<li><strong>比特币</strong>：基础层，就像银行电汇 - 安全但较慢</li>
<li><strong>闪电网络</strong>：快速层，就像信用卡 - 即时且便宜</li>
</ul>

Flash会自动使用两者为您提供最佳体验。了解更多关于[闪电网络](/zh/lightning-network)。

</details>

<details class="faq-item">
<summary><strong>我可以将比特币发送到其他钱包吗？</strong></summary>

是的！Flash支持发送到：

<ul>
<li>其他Flash用户（免费且即时）</li>
<li>任何比特币地址（链上）</li>
<li>闪电发票</li>
<li>提现到银行或实物现金</li>
</ul>

应用会自动检测支付类型。

</details>
